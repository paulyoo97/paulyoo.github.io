Kanto Pokedex

1st Generation Pokemon

Context

Pokemon is a Japanese game and animated series centered around adventure and creatures called Pokemon. The Pokedex project was created to apply data extraction techniques and replicate data that would be found in Pokedex entries from the franchise show. The data utilized is exclusively on 1st Generation Pokemon.

How was the Pokedex built?

The first part of developing this application was the data extraction process. Data and images on each Pokemon were extracted using an automated webscraping bot that was created with Python's Selenium library. Once the relevant data was extracted, the data was exported to an Excel spreedsheet while the images were stored in a local folder. The web application's front-end development was built using Python's Streamilt library. Lastly, the Streamlit cloud platform service was used to host and deploy the web application.

The Application

To try out the web app, you can do so by clicking the following link: https://pokedex.streamlit.app/
