Convolutional Neural Network

Brain Tumor Identification

Context

Using Python's TensorFlow and Keras libraries, a convolutional neural network model was developed to predict and identify the existence of brain tumors. The model was trained using a dataset consisting of MRI scans on three thousand brains. The images below on the left and right are examples of brains with and without a tumor, respectively.

Initial Model Results

The initial CNN model was built using a linear stack of layers consisting of Conv2D, MaxPool2D, Flatten, and Dense layers which resulted in a total of 1605634 parameters. After compiling and training the model, we get the following results regarding loss and accuracy:

Epoch Loss Accuracy Validation Loss Validation Accuracy
1/20 14.1018 0.8115 0.7651 0.9100
20/20 7.2441e-06 1.0 0.2781 0.9700

Based on the results, both the loss and accuracy plots demonstrate convergence. The loss plot converged towards 0 and the accuracy converged close to 1, which are both trends indicating improvement in the model.

Model Performance

To finalize this project, the model was provided with test data to determine the model's prediction capabilities. Below is a confusion matrix demonstrating the model's ability to predict if a brain tumor does or does not exist.

We can see from the dark blue regions of the confusion matrix that 381 out of 400 correct predictions were made. This means that about 95% of the time the model is able to correctly diagnosis the presence of tumor. Though such accuracy is an encouraging result, a greater accuracy would be required if it were to be deployed in the healthcare industry.
