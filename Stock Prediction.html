Stock Prediction: "AAPL"

Apple Stock

Context

This project's focus was to predict an Apple Stock's closing price. The market data was extracted from Yahoo! Finance's API as a dataframe. The dataframe was then transformed into arrays and normalized using the sk-learn library before being utilized for the model's training, testing, and validation phases. Below you can see the real world trend of the Apple stock's closing price between January 1,2012 to December 18,2019.

Model Results

The machine learning model was created using the Sequential model with LSTM and Dense layers. After compiling and training the model, the model had a root mean squared error (RMSE) loss of 7.6093e-04. Once trained, the model predictions were generated which had a RMSE loss of about 0.23. Finally, the predicted trend was then plotted along with the real world trend to visualize and validate our observation.

Model Performance

To validate the model, the closing stock price of December 18,2019 was determined. The predicted value was about $67.11 while the true value was $69.93. This means the prediction has an accuray of about 96%.
