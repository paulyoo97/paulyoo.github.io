Foresight

Future Stock Web Application

Context

Foresight r-Ω is a project focused on predicting future stock prices on a number of top tech companies- Apple, Alphabet, Microsoft, NVIDIA, and Tesla. The project was built using python dependencies and deployed using Streamlit Cloud services.

How does Foresight work?

The future stock prices are predicted using Prophet by Meta which is an API capable of modeling and predicting time series data. To visualize trends and data, plotly was used to create graphs, data tables, and projections of prices. Streamlit was utilized for front-end development of the web application. Finally, the website was hosted and deployed using the Streamlit cloud platform.

The Application

To try out the web app, you can do so by clicking the following link: https://foresight.streamlit.app/
